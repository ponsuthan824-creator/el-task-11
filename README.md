 How to Run the Program
1. Open *IntelliJ IDEA*
2. Create a *Java Project*
3. Create a class named UserDetailsApp
4. Paste the Java code into the class
5. Run the program
6. Enter Name, Email, and Phone Number
7. Data will be saved in users.txt

 Conclusion
This project demonstrates basic *Java File Handling* concepts using *IO streams*, making it easy to understand for beginners and suitable for academic purposes.
